Indice Titulo Descripcion Ejemplo
1 <!DOCTYPE>

se utiliza para declarar la versión de HTML que se está utilizando en un documento web.
2 <a>

Esta etiqueta define un hipervínculo, que se utiliza para enlazar de una página a otra. El atributo más importante del elemento es el atributo href, que indica el destino del enlace.

3 <abbr>

Esta etiqueta se utiliza para representar una abreviatura o acrónimo en un documento HTML. Proporciona información adicional sobre la abreviatura o acrónimo, generalmente en forma de un título o texto de información al pasar el cursor sobre el elemento.

4 <address>

se utiliza en HTML para marcar información de contacto o información sobre el autor de una página web. Puede contener detalles como direcciones de correo electrónico, números de teléfono, direcciones postales u otra información de contacto relevante.

5 <article>

se utiliza para marcar un contenido independiente y autónomo en una página web. Esta etiqueta se usa típicamente para encapsular contenido que es una unidad completa en sí misma y que se puede distribuir o reutilizar de manera independiente, como artículos de noticias, blogs, publicaciones de foros, comentarios, historias, reseñas, entre otros.

6 <aside>

Por lo general, se utiliza para contenido que está relacionado con el contenido principal pero que puede considerarse separado y no es esencial para la comprensión principal del contenido.

7 <audio>

Puedes utilizar esta etiqueta para reproducir archivos de audio, como canciones, efectos de sonido o grabaciones de voz, directamente en una página web.

8 <b>

Se utiliza para aplicar formato de negrita al texto dentro de un documento HTML.

9 <base>

Se utiliza para especificar una URL base para todos los enlaces relativos dentro de un documento HTML. Esto significa que si tienes varios enlaces relativos en tu documento (enlaces que no comienzan con "http://" o "https://"), la URL base especificada en la etiqueta <base> se utilizará como punto de partida para todas esas URL relativas.

10 <basefront>

Se utiliza para establecer una fuente base para todo el contenido de una página web.

11 <blockquote>

Se utiliza para citar texto de otra fuente o para destacar un bloque de texto que se considera una cita dentro de un documento HTML.

12 <body>

se utiliza para definir el contenido principal de un documento HTML. Todo el contenido visible de una página web, como texto, imágenes, enlaces, videos y más, se coloca dentro de la etiqueta <body>.

13 <br>

Se utiliza para crear un salto de línea dentro de un documento HTML.

14 <button>

se utiliza para crear un botón interactivo en una página web que los usuarios pueden hacer clic para activar una acción o una función específica. Los botones son elementos esenciales para la creación de formularios, envío de datos y la interacción general del usuario en sitios web.

15 <canvas>

se utiliza para crear gráficos y dibujos en una página web utilizando JavaScript. Esta etiqueta proporciona un lienzo en blanco en el cual puedes dibujar gráficos, realizar animaciones, crear gráficos interactivos y mucho más.

16 <caption>

se utiliza dentro de una tabla (elemento <table>) para proporcionar un título o una descripción para la tabla. La etiqueta <caption> debe ubicarse dentro del elemento <table> pero antes de cualquier elemento <tr> que contenga filas de datos.

17 <cite>

Se utiliza para marcar el título de una obra o el nombre de una fuente citada dentro del texto de una página web. Se usa para indicar que un fragmento de texto es una cita o una referencia a una fuente externa, como un libro, un artículo, una película o un sitio web.

18 <code>

se utiliza para marcar fragmentos de texto que representan código de programación o instrucciones, y se muestra en una fuente de tipo de máquina monoespaciada (generalmente en un tipo de letra que es igual en ancho para cada carácter) para distinguirlo del texto regular.

19 <col>

Se utiliza dentro de <colgroup> para definir atributos específicos para una columna o un grupo de columnas.

20 <colgroup>

Se utiliza para agrupar columnas.

21 <datalist>

Se utiliza para proporcionar una lista de opciones predefinidas que pueden vincularse con un elemento <input> de tipo text. Esta etiqueta permite que el usuario vea sugerencias basadas en la entrada mientras escribe en el campo de texto.

22 <dd>

Se utiliza junto con la etiqueta <dt> y la etiqueta <dl> para definir una lista de descripción en HTML.

23 <dl>

se utiliza para crear listas de definición, que son listas de términos y sus respectivas definiciones o descripciones. Esta etiqueta se utiliza comúnmente en documentos donde es necesario proporcionar una lista de términos y sus significados o explicaciones.

24 <dt>

Se utiliza junto con la etiqueta

(lista de definición) para definir un término o título en una lista de definición. Se utiliza para marcar el término que se va a describir o definir utilizando la etiqueta
(elemento de descripción).

25 <del>

se utiliza para representar el texto o contenido que ha sido eliminado o eliminado de un documento. Es comúnmente utilizada en documentos colaborativos o de revisión, como páginas web, para mostrar a los usuarios qué parte del texto ha sido eliminada o modificada. Por lo general, el texto dentro de la etiqueta <del> se muestra con una línea tachada o marcada de alguna manera para indicar que ha sido eliminado.

26 <details>

se utiliza para crear elementos desplegables o plegables que contienen información adicional. Es especialmente útil cuando deseas mostrar información adicional de manera oculta y permitir que los usuarios la muestren o oculten según su elección.

27 <dfn>

se utiliza para definir un término dentro de un documento web, generalmente para proporcionar una explicación o definición de ese término. Se usa para marcar y destacar un término específico en el texto, de manera que se pueda indicar su significado o contexto.

28 <dialog>

e utiliza para crear una ventana emergente o un cuadro de diálogo dentro de una página web. Esta etiqueta se utiliza en combinación con JavaScript para crear interacciones modales, donde el contenido de la ventana emergente se superpone al contenido principal de la página y generalmente requiere que el usuario realice alguna acción antes de continuar.

29 <div>

se utiliza como un elemento contenedor genérico para agrupar y estructurar otros elementos HTML. No tiene un significado semántico propio y se utiliza principalmente con fines de diseño y estilo.

30 <em>

se utiliza para enfatizar o resaltar texto en un documento web. Por defecto, el texto dentro de <em> se muestra en cursiva, pero la principal intención de esta etiqueta es dar énfasis al contenido, no necesariamente aplicar un estilo específico.

31 <fieldset>

Se utiliza para agrupar elementos relacionados en un formulario web y crear un contenedor con un borde alrededor de esos elementos.

32 <figcaption>

se utiliza en combinación con la etiqueta <figure> en HTML para proporcionar una descripción o un título para una figura, como una imagen o un diagrama. La etiqueta <figure> se utiliza para encapsular contenido multimedia, mientras que <figcaption> se coloca dentro de <figure> para describir o proporcionar contexto para ese contenido.

33 <figure>

Se utiliza para encapsular y agrupar contenido multimedia, como imágenes, gráficos, diagramas, videos, etc.

34 <footer>

Se utiliza para definir el pie de página de una página web o una sección específica de una página. El pie de página suele contener información de contacto, enlaces a páginas relacionadas, derechos de autor, información de licencia y otros detalles relevantes para la página o la sección.

35 <form>

Se utiliza para crear formularios web interactivos que permiten a los usuarios enviar datos al servidor web o realizar acciones en la página. Los formularios son esenciales para la recopilación de información, como registros de usuario, comentarios, búsquedas y más.

36 <h1>

Se utiliza para definir un encabezado de nivel 1 en una página web. Los encabezados son elementos utilizados para estructurar y jerarquizar el contenido de una página web, y <h1> es el encabezado de nivel más alto, lo que significa que representa el título principal de la página.

37 <h2>

Se utiliza para definir un encabezado de nivel 2 en una página web. Los encabezados de nivel 2 representan subsecciones importantes o títulos secundarios dentro de una página y se utilizan para estructurar y jerarquizar el contenido.

38 <h3>

Se utiliza para definir un encabezado de nivel 3 en una página web. Los encabezados de nivel 3 representan subsecciones aún más específicas o títulos terciarios dentro de una página y se utilizan para estructurar y jerarquizar el contenido en un nivel inferior al <h2>.

39 <h4>

Se utiliza para definir un encabezado de nivel 4 en una página web. Los encabezados de nivel 4 representan subsecciones aún más específicas o títulos cuaternarios dentro de una página, y se utilizan para estructurar y jerarquizar el contenido en un nivel inferior al <h3>.

40 <h5>

Se utiliza para definir un encabezado de nivel 5 en una página web. Los encabezados de nivel 5 representan subsecciones aún más específicas o títulos quinarios dentro de una página y se utilizan para estructurar y jerarquizar el contenido en un nivel inferior al <h4>.

41 <h6>

Se utiliza para definir un encabezado de nivel 6 en una página web. Los encabezados de nivel 6 son los más bajos en la jerarquía de encabezados y se utilizan para representar subsecciones extremadamente específicas o títulos sextenarios dentro de una página.

42 <head>

Se utiliza para proporcionar información y metadatos sobre la página web, como su título, enlaces a hojas de estilo, enlaces a scripts, configuración de codificación de caracteres y más. El contenido dentro de la etiqueta <head> no se muestra directamente en la página visible por el usuario, sino que se utiliza para configurar y describir la página.

43 <header>

Se utiliza para definir la cabecera de una sección o de toda una página web. El contenido dentro de <header> suele incluir elementos como el logotipo del sitio web, el título de la página, la navegación principal, y otros elementos importantes que deben aparecer en la parte superior de la página o de una sección específica.

44 <hgroup>

Se utiliza para agrupar los elementos de encabezado <h1> y <h2>. Esto indicaría que "Título principal" es el título más importante y "Subtítulo" es un subtítulo relacionado.

45 <hr>

Se utiliza para crear una línea horizontal o un "separador" en una página web. Por lo general, se utiliza para dividir visualmente el contenido de una página en secciones o para marcar una separación clara entre elementos del contenido.

46 <html>

Se utiliza para definir el comienzo y el final de un documento HTML. Es la envoltura principal que rodea todo el contenido de una página web y especifica que el documento está escrito en HTML.

47 <i>

Se utiliza para aplicar formato de "itálica" al texto dentro de ella.

48 <iframe>

Se utiliza para incrustar otro documento HTML dentro de la página actual. Esta etiqueta permite mostrar contenido de una página web en el interior de otra, como por ejemplo, un video de YouTube, un mapa de Google Maps o incluso otro sitio web completo.

49 <img>

Se utiliza para mostrar imágenes en una página web. Esta etiqueta permite incrustar imágenes en un documento HTML para que los usuarios puedan verlas cuando visitan una página.

50 <input>

Se utiliza para crear controles interactivos en formularios web, que permiten a los usuarios ingresar datos o realizar selecciones. Esta etiqueta puede tener varios tipos de atributos para especificar el tipo de entrada que se va a crear, como campos de texto, botones de opción, casillas de verificación y más.

51 <ins>

para marcar el texto que ha sido insertado o agregado en un documento, en comparación con la versión anterior del mismo documento. Por lo general, se usa en combinación con la etiqueta <del> (para el texto eliminado) para indicar revisiones o cambios en el contenido.

52 <kbd>

Se utiliza para indicar que un texto debe ser interpretado como una entrada de teclado o una tecla presionada en un documento web. Esta etiqueta se utiliza principalmente para representar comandos de teclado, atajos de teclado u otra entrada de teclado en contenido relacionado con informática o tecnología.

53 <label>

Se utiliza para asociar una etiqueta descriptiva a un elemento de formulario, como un campo de entrada (<input>) o un elemento de selección (<select>). Esto ayuda a mejorar la accesibilidad y la usabilidad de un formulario, ya que permite a los usuarios hacer clic en la etiqueta para activar el elemento relacionado, en lugar de tener que hacer clic en el elemento en sí.

54 <legend>

Se utiliza en conjunto con la etiqueta <fieldset> para proporcionar una leyenda o título descriptivo para un grupo de elementos de formulario. El elemento <fieldset> se utiliza para agrupar elementos relacionados dentro de un formulario, como un conjunto de botones de opción o casillas de verificación. La etiqueta <legend> se coloca dentro de <fieldset> y proporciona una descripción significativa de ese grupo de elementos.

55 <li>

Se utiliza para crear elementos de lista en una página web. La etiqueta <li> se usa dentro de elementos de lista, como <ul> (lista desordenada) o <ol> (lista ordenada), para definir elementos individuales de la lista.

56 <link>

Se utiliza para vincular recursos externos, como hojas de estilo (CSS), fuentes, iconos y otros archivos relacionados, con una página HTML. Esta etiqueta se coloca en el encabezado (<head>) del documento HTML y se utiliza para establecer conexiones con recursos externos que afectan la presentación o el comportamiento de la página web.

57 <main>

Se utiliza para marcar el contenido principal o central de una página web. Esta etiqueta se introdujo en HTML5 para proporcionar una forma semántica de identificar el contenido principal de una página, lo que mejora la accesibilidad y la estructura del documento HTML.

58 <mark>

Se utiliza para resaltar o marcar porciones de texto dentro de un documento HTML para llamar la atención del usuario. Por lo general, se usa para resaltar términos de búsqueda o información importante en el texto. El texto marcado generalmente se mostrará con un fondo amarillo por defecto, pero su apariencia exacta puede ser personalizada a través de CSS.

59 <menu>

Se utiliza para definir una lista de comandos o enlaces que representan un menú de navegación o una lista de opciones en una página web. Esta etiqueta se utiliza en combinación con elementos de menú, como <li> (elemento de lista) y <menuitem> (elemento de menú de contexto), para crear menús contextuales, menús de navegación u otros tipos de listas de opciones.

60 <menuitem>

Se utiliza para definir un elemento de menú de contexto en un menú emergente. Estos menús contextuales son comunes en aplicaciones web y se muestran cuando un usuario hace clic con el botón derecho del ratón en un elemento de la página, como una imagen o un enlace.

61 <ol>

Se utiliza para crear listas ordenadas en una página web. Las listas ordenadas se utilizan cuando deseas presentar una lista de elementos en un orden específico, como una lista numerada. Los elementos de una lista ordenada se muestran con números, letras o viñetas por defecto, y se enumeran en orden secuencial.

62 <ul>

Se utiliza para crear listas desordenadas en una página web. Las listas desordenadas se utilizan cuando deseas presentar una lista de elementos que no siguen un orden específico y se enumeran generalmente con viñetas por defecto.

63 <meta>

Se utiliza para proporcionar metadatos o información adicional sobre un documento HTML. Estos metadatos no se muestran directamente en la página web, pero son útiles para los motores de búsqueda, navegadores web y otras herramientas que procesan o interpretan la página.

64 <meter>

Se utiliza para representar una medida o valor en una escala dentro de un rango conocido. Esta etiqueta es especialmente útil para mostrar gráficamente valores que se pueden medir, como el progreso, el nivel de batería, la temperatura, la puntuación de un examen, entre otros.

65 <nav>

Se utiliza para definir una sección de navegación en una página web. Es una forma semántica de marcar y estructurar la navegación de un sitio web, lo que facilita la comprensión de su contenido por parte de los navegadores y las tecnologías de asistencia, como lectores de pantalla. La etiqueta <nav> se usa típicamente para agrupar enlaces o elementos de navegación, como menús, barras de navegación y otros elementos relacionados con la navegación.

66 <noscript>

Se utiliza para proporcionar contenido alternativo que se mostrará en caso de que el navegador del usuario no admita JavaScript o si JavaScript está deshabilitado en el navegador. Esto permite que los desarrolladores web ofrezcan una experiencia básica o alternativa a los usuarios que no pueden o no desean ejecutar scripts en sus navegadores.

67 <optgroup>

Se utiliza para crear grupos de opciones dentro de un menú desplegable (<select>). Esto permite organizar las opciones en categorías o secciones, lo que facilita la navegación y la comprensión del usuario cuando hay muchas opciones relacionadas.

68 <option>

Se utiliza dentro de un elemento <select> para definir las opciones disponibles en un menú desplegable. Cada <option> representa una elección que un usuario puede hacer en el menú desplegable.

69 <output>

Se utiliza para mostrar el resultado de un cálculo o una operación realizada en un formulario HTML. Es especialmente útil cuando se trabaja con formularios interactivos y se desea mostrar el resultado de una acción o cálculo en la misma página.

70 <p>

Se utiliza para definir párrafos de texto en una página web. Sirve para separar y estructurar el contenido en párrafos individuales, lo que hace que el texto sea más legible y organizado.

71 <pre>

Se utiliza para representar texto preformateado en una página web. Esto significa que el texto dentro de una etiqueta <pre> se muestra exactamente como está escrito en el código fuente, incluyendo espacios en blanco, saltos de línea y espacios adicionales. Es útil cuando necesitas mostrar código fuente, texto con formato específico o cualquier contenido en el que la disposición y la indentación sean importantes.

72 <progress>

Se utiliza para crear una barra de progreso que representa el avance o el estado de finalización de una tarea en una página web. Es especialmente útil cuando deseas mostrar visualmente cuánto ha progresado una tarea o cuánto queda por completar.

73 <q>

Se utiliza para indicar una cita corta en un documento HTML. Esta etiqueta está diseñada para citas breves que se incorporan dentro del texto principal de una página web. Por lo general, las citas entre <q> se mostrarán con comillas alrededor del texto citado.

74 <rp>

Se utiliza junto con la etiqueta <ruby> para proporcionar texto de respaldo que se muestra en navegadores que no admiten la notación ruby. La notación ruby se utiliza principalmente en páginas web escritas en idiomas asiáticos como el japonés para indicar pronunciación o significado de caracteres. El contenido dentro de <rp> se mostrará cuando el navegador no pueda interpretar la notación ruby.

75 <rt>

Se utiliza junto con la etiqueta <ruby> y se emplea en la notación ruby, que es comúnmente utilizada en idiomas asiáticos como el japonés para indicar la pronunciación o el significado de caracteres o palabras. El contenido dentro de la etiqueta <rt> representa el texto ruby, que es una guía fonética que se muestra junto a los caracteres originales.

76 <ruby>

Se utiliza para la notación ruby, que se utiliza principalmente en idiomas asiáticos como el japonés para proporcionar una guía fonética que indica la pronunciación o el significado de caracteres o palabras. La notación ruby es especialmente útil cuando se utilizan caracteres complejos y se necesita ayuda para comprender su pronunciación.

77 <s>

Se utiliza para indicar texto que ha sido tachado o eliminado, generalmente para mostrar que es obsoleto o incorrecto. Esta etiqueta se utiliza para aplicar un estilo de línea horizontal a través del texto para indicar que ha sido marcado como incorrecto o que ya no es relevante.

78 <samp>

Se utiliza para marcar texto que representa una muestra de salida o resultados de un programa de computadora, un comando o una entrada de usuario. Es especialmente útil cuando deseas mostrar ejemplos de salida de un programa o cómo se vería la salida de un comando en la línea de comandos.

79 <script>

Se utiliza para incluir código JavaScript en una página web. JavaScript es un lenguaje de programación que se ejecuta en el navegador web del usuario y se utiliza para agregar interactividad, funcionalidad y dinamismo a las páginas web.

80 <section>

Se utiliza para definir secciones temáticas o agrupar contenido relacionado en una página web. Sirve para estructurar y organizar el contenido de una manera semántica y proporcionar una mayor claridad y significado al diseño de la página.

81 <select>

Se utiliza para crear listas desplegables o menús desplegables en una página web. Los menús desplegables permiten a los usuarios seleccionar una opción de una lista de opciones disponibles. Esta etiqueta se utiliza comúnmente en formularios web para permitir que los usuarios hagan selecciones de una manera fácil y estructurada.

82 <small>

Se utiliza para reducir el tamaño del texto y presentar contenido en un tamaño de fuente más pequeño en relación con el texto circundante. A menudo, se utiliza para mostrar texto que es menos importante o para agregar notas de pie de página u otros detalles de menor relevancia.

83 <span>

Se utiliza para aplicar estilos o manipulaciones específicas a una parte del texto dentro de un elemento HTML sin cambiar el significado o la estructura del texto. En otras palabras, <span> se utiliza para etiquetar o agrupar una sección de texto en un elemento más grande para aplicar estilos CSS o realizar manipulaciones con JavaScript.

84 <strong>

Se utiliza para dar énfasis a un texto y hacerlo visualmente más destacado en negrita en comparación con el texto circundante. A menudo se interpreta como un énfasis fuerte o una importancia especial en el texto.

85 <style>

Se utiliza para incluir reglas de estilo CSS (Cascading Style Sheets) en una página web. Con esta etiqueta, puedes definir y aplicar estilos CSS directamente en el documento HTML, lo que permite controlar la apariencia y el diseño de los elementos HTML en la página.

86 <sub>

Se utiliza para representar texto subíndice en una página web. El texto subíndice se muestra ligeramente más abajo y a una escala más pequeña que el texto circundante. Esto se usa comúnmente para representar fórmulas químicas, notación matemática, números ordinales, marcas de registro y otros elementos que requieren un posicionamiento de texto más bajo en relación con el texto principal.

87 <summary>

Se utiliza para definir un resumen o título corto para un elemento <details>. El elemento <details> se utiliza para crear un área desplegable en una página web que puede ocultar o mostrar contenido adicional cuando se hace clic en el resumen proporcionado. La etiqueta <summary> se coloca dentro del elemento <details> y proporciona el texto que se mostrará como el encabezado o título del área desplegable.

88 <sup>

Se utiliza para crear texto sobrescrito, lo que significa que el texto contenido dentro de esta etiqueta se mostrará en una posición ligeramente elevada en relación con el texto circundante. Se utiliza comúnmente para representar exponentes, números ordinales, marcas de registro y otros casos en los que el texto debe aparecer en una posición superior.

89 <svg>

Se utiliza para incorporar gráficos vectoriales escalables en una página web. SVG significa "Scalable Vector Graphics" (Gráficos Vectoriales Escalables), y es un formato de archivo gráfico que permite representar imágenes de manera precisa y nítida a cualquier tamaño sin pérdida de calidad.

90 <table>

Se utiliza para crear tablas de datos en una página web. Las tablas son una forma efectiva de organizar información en filas y columnas, lo que facilita la presentación de datos de manera estructurada y legible. La etiqueta <table> se utiliza junto con otras etiquetas como <tr> (fila), <th> (encabezado de columna) y <td> (celda de datos) para definir la estructura y el contenido de la tabla.

91 <tbody>

Se utiliza para definir la sección de cuerpo de una tabla (<table>).

92 <td>

Se utiliza para crear celdas de datos en una tabla (<table>). Cada celda (<td>) contiene información o datos específicos en una fila y columna determinadas de la tabla. Las celdas de datos son los elementos principales que almacenan y presentan la información tabular en una página web.

93 <tfoot>

Se utiliza para definir la sección de pie de una tabla (<table>). El pie de la tabla generalmente se utiliza para agregar resúmenes, totales u otra información relevante que deba aparecer en la parte inferior de una tabla.

94 <th>

Se utiliza para crear celdas de encabezado en una tabla (<table>). Los encabezados de tabla se utilizan para identificar y describir las columnas o filas de la tabla, lo que facilita la comprensión y la navegación de la información en la tabla.

95 <thead>

Se utiliza para definir la sección de encabezado de una tabla (<table>). Los encabezados de tabla se utilizan para proporcionar etiquetas o descripciones para las columnas o filas de la tabla, lo que facilita la comprensión de los datos en la tabla. La etiqueta <thead> se utiliza comúnmente en combinación con las etiquetas <th> (celdas de encabezado) para crear encabezados claros y legibles en una tabla.

96 <textarea>

Se utiliza para crear un área de texto de múltiples líneas en un formulario web. Proporciona a los usuarios un espacio para ingresar texto extenso, como comentarios, descripciones o cualquier tipo de texto que requiera más que una sola línea de entrada. Los elementos <textarea> son especialmente útiles cuando se necesita recopilar información de texto más larga que la que puede acomodar un campo de entrada de una sola línea (<input type="text">).

97 <time>

Se utiliza para representar información sobre una fecha y/u hora, lo que facilita la identificación y el formateo de datos temporales en una página web. Esta etiqueta ayuda a los motores de búsqueda y a las herramientas de accesibilidad a comprender y procesar la información temporal de manera más precisa.

98 <title>

Se utiliza para definir el título de una página web. El texto que se coloca dentro de la etiqueta <title> aparece en la barra de título del navegador web cuando se abre la página y también se muestra como el título de la pestaña en la que se encuentra la página.

99 <tr>

Se utiliza para definir una fila (row) en una tabla (<table>). En una tabla, cada fila contiene celdas de datos <td> que están organizadas en columnas. La etiqueta <tr> se utiliza para agrupar y estructurar estas celdas de datos en una fila específica. Cada fila en una tabla generalmente contiene el mismo número de celdas (aunque esto no es estrictamente necesario).

100 <var>

Se utiliza para definir una variable o una expresión matemática en un documento web. Esta etiqueta se usa para destacar y formatear visualmente variables, fórmulas matemáticas o cualquier otro tipo de contenido que represente valores variables en un contexto matemático o de programación.

101 <video>

Se utiliza para incorporar contenido de video en una página web. Permite a los desarrolladores web embeber videos directamente en una página sin necesidad de usar reproductores externos o complementos de terceros. Esto facilita la reproducción de videos en línea sin depender de aplicaciones externas.

102 <wbr>

Se utiliza para indicar un punto de inserción de una posible división de una palabra o secuencia de caracteres en una línea. "WBR" significa "Word Break Opportunity" (Oportunidad de ruptura de palabra). La etiqueta <wbr> permite que el navegador web decida dónde y cuándo dividir una palabra larga o una secuencia de caracteres en varias líneas si el espacio horizontal es limitado, lo que ayuda a mejorar la legibilidad de un texto largo.
