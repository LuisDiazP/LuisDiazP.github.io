Resumen de AJAX

¿Qué es ajax?

AJAX (Asynchronous JavaScript and XML) es una técnica de programación que permite actualizar el contenido de una página web de manera asincrónica, es decir, sin necesidad de recargar por completo la página. Emplea un conjunto de tecnologías como JavaScript, XML o JSON, HTML y CSS para facilitar la comunicación entre el cliente y el servidor.

Importancia y Propósito:

La importancia de AJAX radica en su capacidad para mejorar la experiencia del usuario al permitir la actualización dinámica de contenidos sin interrumpir la navegación. Su propósito principal es brindar una experiencia más fluida y rápida en las aplicaciones web, al realizar solicitudes al servidor en segundo plano, lo que reduce el tiempo de espera y ofrece interfaces más interactivas.

Características:

  1. Asincronía: Permite realizar solicitudes al servidor sin interrumpir la interacción del usuario con la página.
  2. Actualización parcial: Puede actualizar partes específicas de una página sin recargarla por completo.
  3. Tecnologías múltiples: Utiliza diferentes tecnologías como JavaScript, XML o JSON, HTML y CSS para su funcionamiento.
  4. Ahorro de ancho de banda: Al actualizar solo partes específicas, reduce la cantidad de datos transferidos entre el cliente y el servidor.

Uso de AJAX

Se utiliza en una amplia gama de aplicaciones web para mejorar la experiencia del usuario. Ejemplos comunes incluyen formularios que validan datos en tiempo real, comentarios que se actualizan sin recargar la página, carga dinámica de contenido al hacer scroll, entre otros. Plataformas como Gmail, Google Maps y redes sociales hacen uso extensivo de AJAX para ofrecer interacciones fluidas y rápidas sin recargar por completo la página web.



Ejemplo

AJAX 
                        
                        document.addEventListener("DOMContentLoaded", function() {
                            document.getElementById("cargarUsuarios").addEventListener("click", function() {
                                var xhr = new XMLHttpRequest();
                        
                                xhr.onreadystatechange = function() {
                                if (xhr.readyState === XMLHttpRequest.DONE) {
                                    if (xhr.status === 200) {
                                    var usuarios = JSON.parse(xhr.responseText);
                                    mostrarUsuarios(usuarios);
                                    } else {
                                    console.error('Hubo un error al cargar los usuarios.');
                                    }
                                }
                                };
                        
                                xhr.open("GET", "usuarios.json", true);
                                xhr.send();
                            });
                        
                            function mostrarUsuarios(usuarios) {
                                var usuariosDiv = document.getElementById("usuarios");
                                usuarios.forEach(function(usuario) {
                                var usuarioDiv = document.createElement("div");
                                usuarioDiv.textContent = `Nombre: ${usuario.nombre}, Email: ${usuario.email}`;
                                usuariosDiv.appendChild(usuarioDiv);
                                });
                            }
                            });
